3.《最少必要编程课》

任何人都能学会的编程基础知识课。

0.写在前面​

你总不能只靠一本书，一套教程就学会编程吧？

让0基础的同学可以掌握最少必要编程知识，以便未来可以进一步学习编程。

下面，从盲从开始，一步步点亮你的技能树，让编程知识不知不觉在大脑中完成构建。

1.编程环境准备​

  • 前置知识：会下载并安装软件。
  • 学习目标：安装Python并运行第一行Python代码。

1.1.安装Python​

  1. 访问 Python官方下载页面，然后直接点击“Download Python 3.11.2”开始下载Python安装包，版本号可能不同，下载页面上的最新版即可：

  2. 点击另存为，将Python安装包保存在电脑上，比如选择“桌面”：

  3. 如果系统弹出如下提示，选择运行即可：

  1. 注意：勾选“Add python.exe to PATH”，这项默认是不勾选的，然后点击“Install Now”，开始安装Python：

  2. 点击“Disable path length limit”，然后点击右下角的“Close”完成Python安装：

1.2.第一行代码​

  1. 打开PowerShell：

    按键盘上的“Win”键，或点击 Windows 任务栏的“开始”按钮，然后输入powershell，即可找到 Windows PowerShell，点击打开Windows PowerShell

    Windows PowerShell是Windows上的命令行工具，或者叫终端，区别于我们平时通过图形化界面，通过鼠标点击操纵计算机，Windows PowerShell是一个通过输入“命令”的方式操纵计算机的工具。

  2. 运行Python解释器：

    Python解释器，就是Python自带的运行Python代码的地方，在打开的 Windows PowerShell 窗口中输入python，看到如下图结果，包含Python的版本信息，3.11.4：

  3. 运行print("Hello World")

    在Python解释器内输入print("Hello World")，得到如下的结果，简单吧，你已经成功写了一行Python代码并执行。

    这行代码的功能是将print()括号里的内容Hello World“打印”（显示）在电脑屏幕上。

    其中：

    • >>>，这三个连续的大于号称为提示符，表示我们可以在后面输入代码；
    • "Hello World"所在的行前面是没有任何符号的，表示是解释器运行代码的结果。

1.3.退出Python解释器​

当我们在Python解释器运行完Python代码时，如何退出Python解释器呢？

其实，可以直接将PowerShell窗口关掉；

当然也可以通过在解释器中输入exit()然后回车，会发现PowerShell中的提示符（每行最前面的字符）由>>>变为了PS，这也就意味着从Python解释器退出到了PowerShell。

这里小结一下：

  • Windows PowerShell是Windows的命令行（终端）工具，是我们通过命令的方式跟Windows打交道的软件；

  • 在Windows PowerShell中，通过命令python，就可以打开Python解释器这个程序；

  • 在Python解释器内，输入exit()，就是通过命令的方式，关闭Python解释器，回到Windows PowerShell中。

1.4.作业​

  1. 在Python解释器中运行print("Hello World")

1.5.总结​

这节课，我们：

  • 安装了Python
  • 知道怎么打开Windows PowerShell了
  • 在Power Shell中运行了Python解释器
  • 在Python解释其中运行了代码：print("Hello World")
  • 退出Python解释器。

1.5.技能树​

目前为止，您具备了运行代码的环境，并运行了代码。

2.值​

  • 前置知识：在Python解释器中成功运行代码：print("Hello World")
  • 学习目标：在Python解释器中输入多种类型的 值 ，如：数字，文字。

2.1.整数​

比如0，正整数 和 复整数，如：

0

1, 2, 3, ...

-1, -2, -3, ...

2.2.浮点数（小数）​

比如正小数，复小数，如：

1.1, 2.1, 3.1, ...

-1.1, -2.1, -3.1, ...

2.3.字符​

指单个字符。

可以是英文，中文，标点符号，特殊符号等：

注意，字符需要包裹在英文的单引号'或英文双引号"里，如：

"a", "b", "c", "你", "好","@"

2.4.字符串​

一连串字符，注意，字符串需要包裹在英文的单引号'或英文双引号"里，如：

"abc你好@"

学编程的第一行代码print("Hello World")中的"Hello World"也是字符串，尤其要指出的，Hello和World之间的空格，也是字符。

留意区别：2.3的例子中，每个字符都包含在双引号"中，2.4的例子字符串整体包含在双引号"中。

2.6.总结​

这节课，我们：

  • 在Python解释器中输入了
    • 整数
    • 浮点数（小数）
    • 字符和字符串

2.7.技能树​

目前为止，您在编程环境中，输入了多种类型的值，这些都是我们未来代码中的元素。

3.变量​

  • 前置知识：在Python解释器中成功输入多种类型的值。
  • 学习目标：学会给变量赋值，使用变量。

3.1.变量​

上学的时候我们学过一些简单的方程式，比如：

a = 1

b = 2

我们将1赋值给a，2赋值给b;

x= a + b就得到了x = 3;

y = a * b就得到了y = 2。

很显然，我们只需要修改a的值，就可以得到新的 x 和 y 的值。

在编程中，也是一样的，我们可以修改 a 和 b 的值，代码中所有用到 a 和 b 值的地方就跟着变了。

我们称 a , b为变量，即用来装 值 的容器。

而这个等号=被称作 赋值操作符。

赋值，就是：变量在左边，值在右边，通过等号=连接。

在学习 “第一行代码“ 的时候，我们通过print("Hello World")将Hello World输出到了屏幕上。

根据上面的知识，我们就可以把Hello World赋值给一个变量，然后将这个变量所代表的值输出到屏幕上：

修改后的代码如下：

msg = "Hello World"
print(msg)

上面的代码，我们将Hello World赋值给了变量msg，然后通过print(msg)把变量msg所代表的Hello World输出到了屏幕上。

以后，当我们想修改输出到屏幕上的内容，只需要改变msg的值就可以了，即通过 再次赋值 的方式，让变量等于新的值。

比如：

msg = "Good Morning"
print(msg)

屏幕上就会输出Good Morning。

3.2.作业​

  1. 分别将一个整数，浮点数，字符以及字符串，通过变量输出到屏幕上。

3.3.总结​

这节课，我们：

  • 通过数学公式的类比，理解了赋值及用途；
  • 通过将整数，浮点数，字符以及字符串赋值给一个变量的作业回顾了这些类型的值。

3.4.技能树​

目前为止，您已经掌握了如何给变量赋值，也理解了，未来只要对变量重新赋值，所有用到变量的地方就变成了新值。
