2.小白慢爬营

小白慢爬营 是一个编程学习社群，是编程小白们自学成长的聚集地。

而本网站则是小白慢爬营社群的唯一且官方网站，汇整了关于社群内的一些资讯、活动和项目介绍，是小白慢爬营对外的统一窗口。

本网站总共分以下五大模块。

  • 小白入门
  • 找人同行
  • 编程课
  • 实战进阶
  • 联系

整体脉络是根据每个人自学路上会经历的几个阶段进行延展，并提供相应的解决思路或者方法。

从启动自学、到自学路上遇到障碍想要放弃需要找人同行、以及通过英语小助手等小项目练习强化最少必要编程知识，最后到跨过学进入用的实战项目。

接下来，我们将逐一介绍每个板块的内容。

一、小白入门​

这一部分，是针对完全零基础的纯小白们，或许因为AI 人工智能的刺激，或许是基于成长的需求，你抱着了解一下、初窥一下编程世界的想法，想知道编程是怎么回事，而自己又是否能学的会。

如果你是这部分人群，那么你可以先直接查阅小白入门这一板块，它从编程入门的基础学习资料、如何搭建编程环境、网络环境、最少必要知识点有哪些分别做了介绍。

这一板块主要是试图回答编程如何入门这一问题，达到扫盲的目的。

二、找人同行 ​

编程不能速成，它的学习可能需要几年甚至很多年的时间。而编程自学，更是难上加难。

在编程自学的路上，最容易遭遇的就是中途放弃且无数次的放弃，但只要放弃再想重新拾起可能性就很小。

所以，有时需要借助他人裹挟的力量，需要有人跟我们一起前行，互相扶持、互相鼓励，在自己想要放弃时拽自己一把。

而找人同行 这个板块，就是社群为了解决这个问题而发起的几个活动。

  • 对赌共学
  • 共读
  • 周会

如果你想找人同行，想了解关于这几个活动的具体内容，你可以直接进入并查阅找人同行 这个板块。

三、编程课​

在你一个人或者与人同行，坚持学习并掌握了最少必要编程知识外，你又来到了另一个阶段：

即把书本上的理论知识与现实生活挂钩，练习使用这些基础知识点，让他们从书本上“死”的知识化成身边“活”的可见的东西。

你可以找一些书本上的小项目或者习题进行练习自测，你也可以来到我们的第三板块:编程课

这部分编程课细分为两个部分。

1. 公开课程​

这部分课程是免费内容，是关于 Markdown 的极简教程和 使用Github参与创作， 这两部分知识属于编程者们必备的基础技能，目前社群免费公开。

2. 付费课程​

这部分付费课程， 是社群项目组（主要内容创作者Mumu）针对编程最少必要知识点，独家定制开发的《英语学习小助手机器人》的实战项目。这是一个有实际市场需求和应用场景的机器人开发项目，采用了视频实录、逐字稿的内容形式以及Github Isuues问题提问与答疑的方式进行答疑解惑，同时将问题进行沉淀。

或许，你遇到的问题，走在你前面的同行者已经遇到过了呢，所以，遇到问题，可以到Issues库去搜索，也可以向项目组提出问题，有专门的技术组进行答疑。

关于这部分的具体内容，如果您有兴趣，可以直接进入付费课程进行查阅，里边已经详细的罗列了此课程的列表，你可以从列表中了解内容是否符合您的需求。

四、实战进阶​

编程自学的路上，会遇到很多障碍，除了前面说的入门难、坚持难，最难的是如何从学跨越到用的阶段。

很多人面临同一个问题：

我学了这么久，一年、两年，甚至很多年，我能看懂别人的代码，可是我学的这些知识，究竟能用来干什么呢？

很多人，会被困在这个迷障里，跳不出来，而且理论知识学的越久，越觉得自己学的还不够，因为编程的知识库实在太庞大了。

可是，可是，我以过来人的经验告诉你，这是一个误区：

其实用编程，不需要太多的知识，只要掌握最少必要知识就已足够，剩下的就是拿着生活中“活生生”的项目进行实战就可以，通过一个又一个的实战项目来练手，培养自己的“手感”。这一步，对于从学跨到用，真正用编程，非常重要。

只要我们跟随着，做过一个或者多个实战项目，我们就会发现，我们掌握的理论知识，究竟应该怎么理解。

而实战进阶这个板块，就是为了实现从学到用、练习手感这个目的而存在的。

是的，编程也像其他手艺一样，需要手感。

而手感的练习就需要通过一个一个的项目，如果我们自己有资源、有能力，我们可以自行练习，但如果你也被困在这里很久，感觉找不到出路，不妨来了解一下实战进阶里的实战项目。

这个板块，也是一个付费项目，而且是一个有效期长达7年的付费项目。社群将保持内容更新，从2023年6月3日开始，至少7年。

这里边所有的项目，都不是市场上那些编程培训里提供的“练习题”，而是真正基于生活场景和使用场景，从无到有搭建起来的实战项目。

目前，项目处于第一期招生中，已放出的项目有3项，具体内容可查阅实战进阶板块。

我会在这里等你，陪你一起走过接下来的7年，因为我也是这个项目组的一名成员。

想到，有一批真正的长期主义者，一起在编程自学的路上同行7年，不禁让人心生喜悦呢。

五、联系​

我们目前有基于微信生态的微信社群和基于Mixin生态的Mixin社群，您可根据您的实际情况选择。

如果，您已有Mixin APP，建议您直接入Mixin群。
