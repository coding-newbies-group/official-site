5.6 字符串的索引​

因为字符串是有序容器，所谓有序，就是字符串里的每一个字符都有对应的编号。

索引，其实就是根据字符串的编号进行单个或者多个提取的操作。

索引编号：

  • 第一个字符串的索引值为0。
  • 索引值也可以为负，当为负时，字符串最后一个字符的索引值为-1。

索引提取方法：

Methods意义
s[index]返回索引值为 index 的那个字符
s[start:]返回从索引值为 start 开始一直到字符串末尾的所有字符
s[start:stop]返回从索引值为 start 开始一直到索引值为 stop 的那个字符之前的所有字符
s[:stop]返回从字符串开头一直到索引值为 stop 的那个字符之前的所有字符
s[start:stop:step]返回从索引值为 start 开始一直到索引值为 stop 的那个字符之前的，以 step 为步长提取的所有字符

5.8.2 搜索与替换​

  • 看字符是否存在，先用in判断；

  • 看字符串是否以某个字符开始或者结束的，str.startswith()/str.endswith();

  • 统计出现的次数，用count();

  • 找到第一次出现的位置，用find()或者index()

  • 找到最后一次出现的位置，用rfind()或者rindex()

Methods意义
str.count(sub[,start[,end]])搜寻子字符串出现次数
str.find(sub[, start[, end]])返回第一次sub 出现的那次的位置，即返回值为第一次字符的索引值
str.rfind(sub[, start[, end]])返回最后一次 sub 出现的那次的位置，即返回值为最后字符的索引值
str.index(sub[, start[, end]])作用与 find() 相同，但如果没找到的话，会触发 ValueError 异常
str.rindex(sub[, start[, end]])作用与 rfind() 相同，但如果没找到的话，会触发 ValueError 异常
str.replace(old, new[, count])用 new 替换 old，替换 count 个实例，其中，count 这个参数是可选的。
str.expandtabs(tabsize=8)把字符串中的 TAB（\t）替换成空格，默认是替换成 8 个空格
str.startswith(prefix[, start[, end]])用来判断一个 字符串 是否以某个子字符串起始
str.endswith(suffix[, start[, end]])用来判断一个 字符串 是否以某个子字符串结束

因为列表是有序列表，即列表中的每个元素对对应一个索引值，可以根据这个索引值把对应的元素提取出来。

Methods意义
list[index]返回索引值为 index的元素值
list[:]返回整个列表
list[start:]从索引为 start 的值开始直到末尾

del list[index]

这里要注意del的用法，del list[index]是个语句。

  • 根据索引替换

list[start:stop:step] = ["element1","element2"]

  • 特殊说明

因为列表是可变容器，所以以上操作均对原列表本身进行操作，这代表原列表被改变了。

而字符串因为是不可变容器，所以部分函数不能对字符串使用，而且原先的字符串并没有被改写。

  • 列表可用的内建函数

    • len(),max(),min()

  • 列表可用的Methods：

Methods意义
a.sort()排序
a.sort(reverse=True) #reverse 参数，默认是 Falsereverse() 只对当前序列操作，并不返回一个逆序列表；返回值是 None
a.append()在末尾追加一个元素
a.insert(1,"example")在索引 1 的位置插入 'example'
a.extend()在末尾追加一个列表
a.clear()清空序列
b = a.copy()拷贝一个列表,对一个拷贝b操作，不会更改 “原件”a
a = b对 a操作，相等于对 b操作
a.remove('example')去除 'example' 这个元素，如果有多个 'example'，只删除第一个

6.4 集合（Set)​

6.4.1 概念与定义​

  • 集合是无序容器，即其中元素没有索引值；

  • 集合是去重数据容器，即集合中的元素都是没有重复值的；

  • 集合分可变集合（Set），和不可变集合（Frozen Set)。不可变容器是为了节省内存而设计的一种数据容器，同列表与元组的关系。

6.4.2 创建集合​

  • 创建非空集合：用{}，元素用,间隔；

  • 创建空集合： 必须用set()，而不能用{}；

  • 类型转换： 用set()将其他类型的数据转换成集合；但是转换后的集合是去重后的集合。

  • 集合生成器comprehension: set = {x**2 for x in set1}

6.4.3 对集合的操作​

  • 去重：类型转换
  • 判断是否存在：in
  • 集合可用的内建函数：len()/max()/min()
  • 对set删除元素：set.remove()

6.4.4 集合运算​

  • 并集：|
  • 交集：&
  • 差集：-
  • 对称差集：^

6.4.5 集合运算的Methods​

意义操作符MethodsMethods相当于
并集|set.union(*others)set | other | ……
交集&set.interaction(*others)set & other & ……
差集-set.difference(*others)set - other - ……
对称差集^set.symmetric_difference(other)set ^ other

6.4.6 集合的逻辑比较​

Methods意义
set == otherTrue: set 与 other 相同
set != otherTrue: set 与 other 不同
isdisjoint(other)True: set 与 other 非重合；即，set & other == None
issubset(other)即set<=other，True: set 是 other 的子集
set < otherTrue: set 是 other 的真子集，相当于 set <= other && set != other
issuperset(other)即set>=other,True: set 是 other 的超集
set > otherTrue: set 是 other 的真超集，相当于 set >= other && set != other

6.4.7 集合的更新​

Methods意义
add(elem)把 elem 加入集合
remove(elem)从集合中删除 elem；如果集合中不包含该 elem，会产生 KeyError 错误。
discard(elem)如果该元素存在于集合中，删除它。
pop(elem)从集合中删除 elem，并返回 elem 的值，针对空集合做此操作会产生 KeyError 错误。
clear()从集合中删除所有元素。
set.upset(*others)更新 set, 加入 others 中的所有元素；相当于 `set
set.intersection_update(*others)更新 set, 保留同时存在于 set 和所有 others 之中的元素；相当于 set &= other & ..
set.difference_update(*others)更新 set, 删除所有在 others 中存在的元素；相当于 `set -= other
set.symmetric_difference_update(other)更新 set, 只保留存在于 set 或 other 中的元素，但不保留同时存在于 set 和 other 中的元素；注意，该 Method 只接收一个参数。相当于 set ^= other
