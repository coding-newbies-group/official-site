4.网络环境

这部分，好像不应该叫“网络环境”，而应该叫【搜索能力】。

因为写这部分的主要目的是想帮助你能够借助最新搜索工具，搭建最畅通的搜索环境，充分利用互联网这本百科全书。

自学能力中很重要的一项能力是“搜索能力”，这决定了你搜索答案的范围、找到正确答案的概率，以及排除无效干扰项快速找到最优解的效率。

其中，搜索效率和最优答案的获取取决于工具和技巧，而搜索范围取决于“网络环境”。

因此，这部分的主要内容围绕工具、技巧和环境三个部分展开。

一、搜索工具​

我对搜索工具的使用，经历了百度、谷歌和ChatGPT 三个阶段。

用下来的整体感受就是:

由简入奢易， 由奢入俭难。

一旦你经历过后两者的精准、高效，就无法再回退到过去。

所以，如果你还停留在百度阶段，推荐你尽快完成切换升级。

切换谷歌，主要依赖于网络环境，这里文末会有简单说明。

而ChatGPT，因为它已经火出天，所以网上可能会有很多的教程教你如何注册 OpenAI，如果你能自行搞定，那你非常棒，恭喜你。

可如果你因为各种疑难困惑被卡住，被拦在门外，这里我可以推荐一个最简单快捷使用 ChatGPT 的方法。

  1. 下载MixinAPP，下载路径参考:2.Mixin社群 | 小白慢爬营

  2. Mixin内搜索Mixchat机器人：7000104341

这个MixChat是我们社群的项目发起人Mumu自行开发的机器人，是把ChatGPT引入Mixin生态，从而省去个人注册的麻烦，可以直接调用ChatGPT，功能基于OpenAI官方GPT-3.5 Model实现AI对话功能。

  1. 添加机器人后即可以免费体验 ChatGPT。

  • 25,000个单词的问答量大约需要$0.1。

费用非常低，但可以以助你打开新世界的大门，给你一张通往新世界的门票，我认为这是值得的。
