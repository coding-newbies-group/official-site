1.对赌共学

文档库

对赌学习输出成果文档库

一、 活动说明​

简单说，对赌共学就是在项目组的发起下，由社群成员自发组成自学小组，以对赌基金的方式倒逼自己以输出进行输入。

目的是为了解决自己一个人自学路上容易懈怠、容易放弃的问题，以“他律”来代替“自律”。

一方面，是利用人类天生是群体动物，希望获得群体认同的内在需求和动力，利用“他人的力量”来监督约束自己进行学习；

另一方面，是采用“对赌基金”的经济压力，以人类讨厌痛恨“损失”的心理来督促自己完成自学任务，以便完成赌局。

像是007社群的“写满7年去南极”是一样的道理。

二、活动规则​

2.1 活动周期​

我们的活动以每三个月为一个周期，目标是督促每个成员保持学习的热情和持续进步同时三个月的周期又让个人的时间安排变得非常灵活。

第一期：自20230601-20230831，3个月。

2.2 对赌基金​

为了确保每个人的认真参与，我们设立了对赌金，每人需要支付¥199元。

那些不能按时完成输出的成员将会扣除对赌金，而那些完成了输出的成员则有机会平分这些没有完成的人的对赌金。

这样一来，大家都会更加努力地保持学习的动力，为自己和整个社群创造更好的学习环境。

2.3 学习任务​

在这个周期内，每个月你需要至少输出一篇关于学习编程的文章，分享你的学习心得、项目经验、解决问题的方法等等。这样的输出不仅可以帮助你巩固所学知识，还能够帮助其他成员更好地理解和学习。

通过这个对赌激励的方式，我们相信经过筛选后的成员会更加有活力和学习热情。你将有机会 结识志同道合的朋友，一起交流、讨论问题，分享学习资源和技术见解 。无论你是初学者还是有经验的开发者，这个社群都将为你提供一个良好的学习平台，帮助你不断提升技能，拓展编程视野。

学习方向可以根据自己的兴趣点和需求点自发选择，也可以跟对赌的其他小伙伴们组团共同研究同一个方向。

包括但不仅限于：

  1. 操作系统使用技巧
  2. Web开发
  3. 爬虫
  4. 办公自动化
  5. 数据分析
  6. 编程语言
    1. Python
    2. GO
    3. JavaScript

2.4 对赌金返回机制​

跟其他对赌活动不同的是，我们不是在3个月的对赌活动结束后统一结算，而是把整个过程分成三个阶段，进行阶梯式返还，目的是为了降低难度，尽量让参与的人能更好的参与，甚至更容易把共赌金赢回。

所谓阶梯式返回，就是：

> 第一阶段返还¥30元**（15%），第二阶段¥50元**（25%），第三阶段返完（60%）.

为了方便计算每个阶段调整为整月，3个阶段为1期对赌共学活动，每期前一周完成抵押，每月第一周完成上一月达标返还。

以第一期为例： 阶段1:20230601-20230630 阶段2:20230701-20230731 阶段3:20230801-20230831

  • 每个阶段分享至少一遍文章即获得返还；
  • 每完成一个阶段获得一次对赌金返还；
  • 第一次返还约¥30元（15%），第二次返还约¥50元（25%），第三次返完（60%）。

2023年7月第一周完成6月份达标返还； 2023年8月第一周完成7月份达标返还； 2023年9月第一周完成8月份达标返还； 2023年8月最后一周，完成第二期抵押。

2.5 举例说明​

小明在第一个月分享了文章，返还30元；第二个月未能分享文章，无返还；第三个月分享文章，返还50元，总共返还80元。

类似的，小明在第一个月没有分享，没关系，不会损失所有的押金，继续学习和分享就好，第二个月分享可返还30元，第三个月分享可返还50元。

无论你是想加强编程技能，还是希望与其他编程爱好者共同学习成长，我们的活动都欢迎你的加入！

一起努力，共同追求编程的乐趣和成就！
